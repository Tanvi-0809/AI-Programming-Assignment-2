# 1️⃣ Turing Test Simulation

## Description

This program simulates a simplified version of the Turing Test.

### How It Works

- A judge asks a question.
- The system generates:
  - A human-like response
  - A machine-like response
- A heuristic function analyzes keywords and classifies responses as:
  - HUMAN
  - BOT

### AI Concepts Used

- Human vs Machine differentiation
- Heuristic evaluation
- Randomized response generation
